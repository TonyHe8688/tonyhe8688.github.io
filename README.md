# Tony He Embedded Systems Lab

This repository contains the source for the personal technical site:

[https://tonyhe8688.github.io/](https://tonyhe8688.github.io/)

The site collects engineering notes on embedded Linux, Android SBC bring-up, display integration, hardware modules, industrial HMI development, and SBC customization.

## Main Topics

- Embedded Linux board support
- Android single board computer bring-up
- MIPI DSI, LVDS, RGB TFT, HDMI, backlight, and touch integration
- PoE, UART, RTC, RS485, CAN bus, GPIO, and isolated I/O modules
- LVGL and Qt HMI development
- Carrier board customization, production testing, and board revision tracking

The site is built with Hugo and deployed through GitHub Pages.
